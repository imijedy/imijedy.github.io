About Me

Hi, I’m Imre Hajós. I’ve been interested in web design and web programming since I was 14. At first, I learned HTML and CSS than I started to work with PHP. In my free time I created many websites, and I always found the end products extremely motivational. In High School I gained additional knowledge on the basics of programming, then I’ve learned more about the different kinds of programming languages and paradigms in University.

In 2017 I started to work for a web developer company where I had to further develop web applications as well as maintaining them. At the beginning, I worked on the Frontend side and soon after, I got an insight in Backend, therefore I had a few Full Stack jobs as well. On the Frontend side we used React.js and for the older projects native JavaScript. On the Backend side we utilize Spring Boot framework. During my 8 months at this company, I gained experience in solving Fullstack problems, using Git version-control and writing some Unit tests.

Besides that, I've made a web application for my family in connection with documenting costs and revenues and also made some websites for private customers. For the web application I used Spring Boot framework and HTML, JQuery.

In addition to web development, I also tried many other things. For instance, I attempted to make a game in Unity Game Engine, and I was experimenting with developing it on Android platform. Apart from programming, one of my hobbies is couple dancing. I used to dance in a formation until the end of 2018. Another hobby of mine is working with music. It plays a big role in my life. I have been playing guitar for 15 years, and I am also familiar with playing the piano and using studio techniques.
